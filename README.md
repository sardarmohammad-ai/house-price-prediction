# House Price Prediction

The project uses Linear Regression to predict house prices based on size.

## overview
The model is trained on sample data and predicts prices based on input features.

## Technologies
python
Numpy
Scikit-learn
Matplotlob

## How to run
pip install -r requirements.txt
python main.py

## Output
The model predicts house price and display a graph.
